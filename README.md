# meshing
Point renderer of SDF sculpting.

[HashTable Video](https://youtu.be/zhM_QXTWPGQ)

[Async KD Tree Video](https://youtu.be/fSCYRNSLmAw)

Dependencies:
  OpenGL 4.3
  glew
  glfw3
  glm
  clang
  make
  
Building:
	make -j release

Running:
	make run
